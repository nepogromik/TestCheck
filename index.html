Разворачивается одной командой через install.sh

О сайте

Это пример простого статического сайта, который можно установить на свой сервер с помощью автоматического shell-скрипта.

Как используется

  1. Заливаете этот репозиторий на GitHub.
  2. На сервере выполняете одну команду с wget и bash.
  3. Скрипт спрашивает домен и настраивает nginx + SSL.
